# 数据持久化(草案)

## 概述

红色问答可以将收到的私聊消息和群聊消息保存到postgresql数据库，用作备份。

目前为止，红色问答并不会从postgresql数据库中读取数据。

## 配置

在红色问答的配置文件`config.json`中，添加postgresql相关的配置节：

```json
"postgresql": {
    "dbname": "reddat",
    "host": "localhost",
    "password": "123456789",
    "port": 5432,
    "user": "postgres"
}
```

注意：需要重启红色问答才能生效。
